# restart-touchbar

## Install
```bash:#
git clone https://github.com/yuki9431/restart-touchbar.git

cd restart-touchbar/

ln -s $(pwd)/restart-touchbar.sh /usr/local/bin/restart-touchbar
```

## How to Use
```bash:#
restart-touchbar

# Output
Restarted successfully.

```
